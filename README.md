# Intelligent Medicine Box for Medication Management using IoT

## Overview
The Intelligent Medicine Box is an IoT-based healthcare system designed to help patients manage their medication schedules effectively. The system provides timely reminders for medicine intake, monitors medication usage, and reduces the chances of missed or incorrect doses. This project is especially useful for elderly people, patients with chronic illnesses, and individuals with complex medication routines.

## Features
- Automatic medicine reminder using alarms/buzzer
- LED indication for medicine timing
- Real-time monitoring of medicine intake
- Notification system for missed doses
- IoT-based data tracking and management
- User-friendly and reliable medication support

## Working Principle
1. User sets medication schedule in the system.
2. RTC module keeps track of time accurately.
3. When medicine time arrives, the buzzer and LED alert the user.
4. The user takes medicine and confirms intake using a button.
5. Medicine intake data is recorded and monitored through IoT.
6. If a dose is missed, the system sends a notification or warning.
